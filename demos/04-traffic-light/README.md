# 04 - 交通灯模拟

## 实验目标

通过控制红、黄、绿三盏 LED，模拟真实交通灯的工作逻辑。

## 硬件准备

- 树莓派 4B
- LED 3 颗（红、黄、绿）
- 电阻 220Ω ~ 1kΩ 3 个
- 面包板和跳线若干

## 引脚分配

| LED 颜色 | GPIO 编号 | 物理引脚 |
|----------|-----------|----------|
| 红色     | GPIO17   | Pin 11   |
| 黄色     | GPIO18   | Pin 12   |
| 绿色     | GPIO22   | Pin 15   |
| GND      | -        | Pin 6    |

## 接线图

```
[树莓派] 
GPIO17 (Pin 11) ─────── 电阻 ─────── 红色 LED 正极
GPIO18 (Pin 12) ─────── 电阻 ─────── 黄色 LED 正极
GPIO22 (Pin 15) ─────── 电阻 ─────── 绿色 LED 正极
GND     (Pin 6)  ──────────────────  所有 LED 负极（面包板蓝条）
```

## 运行代码

```bash
python demos/04-traffic-light/traffic-light.py
```

## 工作流程

```
红灯亮 5 秒 → 绿灯亮 5 秒 → 黄灯亮 2 秒 → 循环
```

## 关键知识点

1. **多个 GPIO 同时控制**: 同时管理多个输出引脚
2. **时序控制**: 模拟真实交通灯的切换逻辑
3. **状态管理**: 不同灯之间的互斥关系
