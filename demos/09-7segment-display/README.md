# 05 - 数码管显示

## 实验目标

使用 TM1637 数码管模块显示数字、字母和文字。

## 什么是数码管

数码管是一种常见的电子显示器件，由多个发光二极管（LED）组成，可以显示数字、字母和部分符号。

### 4 位数码管模块

本实验使用的 TM1637 模块是一款经济实惠的 4 位 8 段数码管驱动模块：

**模块特点**：
- 4 位 8 段数码管（带小数点）
- 8 级亮度可调
- 供电电压 3.3V-5V
- 仅需 2 根信号线（CLK、DIO）即可控制


## 引脚识别

模块有 4 个引脚，需要正确识别：

### 方法一：查看模块上的印字

大多数 TM1637 模块在 PCB 上印有 GND、VCC、DIO、CLK 标识。找到这些文字后，按顺序连接即可。

### 方法二：常见引脚顺序

如果没有印字，一般常见的引脚顺序（从靠近芯片一端开始）是：

| 引脚 | 说明 |
|------|------|
| GND  | 地线 |
| VCC  | 电源 (3.3V-5V) |
| DIO  | 数据引脚 |
| CLK  | 时钟引脚 |

### 方法三：观察 PCB 走线

- **VCC**: 通常连接到芯片的电源引脚，附近有电容
- **GND**: 连接到芯片的地引脚
- **DIO/CLK**: 连接到芯片的两个控制引脚

> **注意**: 如果接线错误，模块不会显示，但不会烧坏。调整接线直到正常显示即可。

## 引脚分配

| 模块引脚 | GPIO 编号 | 物理引脚 |
|----------|-----------|----------|
| GND      | -         | Pin 6    |
| VCC      | 3.3V      | Pin 1    |
| DIO      | GPIO2     | Pin 3    |
| CLK      | GPIO3     | Pin 5    |

## 接线图

```
[树莓派]           [TM1637 模块]
GPIO3 (Pin 5)  ──── CLK
GPIO2 (Pin 3)  ──── DIO
3.3V  (Pin 1)  ──── VCC
GND    (Pin 6)  ──── GND
```

## 常见问题

**数码管不显示？**
- 检查 VCC 和 GND 是否正确连接
- 检查 CLK、DIO 接线是否正确
- 确认库已正确安装：`pip3 list | grep tm1637`

**只显示乱码？**
- 可能是 DIO 和 CLK 接线反了，尝试交换

**亮度太暗？**
- 调用 `tm.brightness(7)` 设置最高亮度
