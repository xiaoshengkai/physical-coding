# 03 - 流水灯

## 实验目标

控制 4 颗 LED 依次点亮，形成流水效果。

## 硬件准备

- 树莓派 4B
- LED 4 颗
- 电阻 220Ω ~ 1kΩ 4 个
- 面包板和跳线若干

## 连接图
```
      蓝条(-)  (一行孔，横向连通)
    ┌─────────────────────────────────────┐
    │ (-)1 (-)2 (-)3 (-)4 (-)5 (-)6 ...  │   ← 蓝条孔
    └─────────────────────────────────────┘
    ┌─────────────────────────────────────┐
    │ A   B   C   D   E                    │
    ├─────────────────────────────────────┤
1   │ ◉   ◉   ◉   ◉   ◉   ← 电阻R1一端插D1，R1另一端插E1，LED1正极插E1，LED1负极插蓝条(-)1
2   │ ◉   ◉   ◉   ◉   ◉   ← 电阻R2一端插D2，R2另一端插E2，LED2正极插E2，LED2负极插蓝条(-)2
3   │ ◉   ◉   ◉   ◉   ◉   ← 电阻R3一端插D3，R3另一端插E3，LED3正极插E3，LED3负极插蓝条(-)3
4   │ ◉   ◉   ◉   ◉   ◉   ← 电阻R4一端插D4，R4另一端插E4，LED4正极插E4，LED4负极插蓝条(-)4
5   │ ◉   ◉   ◉   ◉   ◉
6   │ ◉   ◉   ◉   ◉   ◉
    └─────────────────────────────────────┘

树莓派连接：
- GPIO17 (引脚11) ──[母对母线A]── 夹住 R1 在 D1 的引脚
- GPIO18 (引脚12) ──[母对母线B]── 夹住 R2 在 D2 的引脚
- GPIO22 (引脚15) ──[母对母线C]── 夹住 R3 在 D3 的引脚
- GPIO23 (引脚16) ──[母对母线D]── 夹住 R4 在 D4 的引脚
- GND (引脚6)     ──[母对母线E]── 夹住 LED1 负极在蓝条(-)1 的引脚（或者单独插一个转接引脚在蓝条上）
```

## 🔄 等效电路
每个 LED 的电路路径：

```
树莓派 GPIO (3.3V 高电平) → 杜邦线 → 电阻 → LED 正极 → LED 负极 → 蓝条 → 杜邦线 → 树莓派 GND
```

## 运行代码

```bash
python demos/03-led-marquee/led-marquee.py
```

## 关键知识点

1. **多 GPIO 控制**: 同时控制多个引脚
2. **列表操作**: 用列表管理多个 LED 对象
3. **循环与反向**: `reversed()` 实现反向流水
