# 10 - 按键控制秒表

## 实验目标

使用两个按键控制数码管秒表，实现开始/暂停和复位功能。

## 硬件准备

- 树莓派 4B
- TM1637 数码管模块
- 两个按键
- 杜邦线 若干

## 软件准备

```bash
# 安装 gpiozero 和 tm1637 库
pip3 install gpiozero tm1637
```

## 引脚分配

### 数码管

| 数码管引脚 | 树莓派 GPIO | 物理引脚 | 说明 |
|------------|-------------|----------|------|
| VCC | 3.3V | Pin 1 | 供电 |
| GND | GND | Pin 6 | 共地 |
| DIO | GPIO2 | Pin 3 | 数据 |
| CLK | GPIO3 | Pin 5 | 时钟 |

### 按键

| 按键 | 树莓派 GPIO | 物理引脚 | 说明 |
|------|-------------|----------|------|
| 按键A（开始/暂停） | GPIO26 | Pin 37 | 另一端接 GND |
| 按键B（复位） | GPIO19 | Pin 35 | 另一端接 GND |

## 运行代码

```bash
python demos/10-stopwatch/main.py
```

或使用快速命令：

```bash
npm run run 10
```

## 操作说明

- **按键1 (GPIO26)**: 开始 / 暂停计时
- **按键2 (GPIO19)**: 复位秒表

## 效果

- 数码管显示当前计时（整数秒，最大 9999）
- 按开始键后计时开始，数码管实时更新
- 按暂停键后计时暂停
- 按复位键后时间归零

## 关键知识点

1. **gpiozero Button**: 树莓派 GPIO 按键库，支持内部上拉
2. **tm1637**: 数码管驱动库
3. **状态机**: 通过布尔变量控制计时状态
