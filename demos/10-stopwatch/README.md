# 10 - 单按钮秒表

## 实验目标

使用一个按键控制数码管秒表，短按开始/暂停，长按复位。

## 硬件准备

- 树莓派 4B
- TM1637 数码管模块
- 一个按键
- 杜邦线 若干

## 软件准备

```bash
# 安装 gpiozero 和 tm1637 库
pip3 install gpiozero tm1637
```

## 引脚分配

### 数码管

| 数码管引脚 | 树莓派 GPIO | 物理引脚 | 说明 |
|------------|-------------|----------|------|
| VCC | 3.3V | Pin 1 | 供电 |
| GND | GND | Pin 6 | 共地 |
| DIO | GPIO2 | Pin 3 | 数据 |
| CLK | GPIO3 | Pin 5 | 时钟 |

### 按键

| 按键 | 树莓派 GPIO | 物理引脚 | 说明 |
|------|-------------|----------|------|
| 按键 | GPIO26 | Pin 37 | 另一端接 GND |

## 运行代码

```bash
python demos/10-stopwatch/main.py
```

或使用快速命令：

```bash
npm run run 10
```

## 操作说明

- **短按 (< 1秒)**: 开始 / 暂停计时
- **长按 (> 1秒)**: 复位秒表

## 效果

- 数码管显示当前计时（整数秒，最大 9999）
- 短按开始键后计时开始，数码管实时更新
- 短按暂停键后计时暂停
- 长按后时间归零

## 关键知识点

1. **gpiozero Button**: 树莓派 GPIO 按键库，支持内部上拉
2. **tm1637**: 数码管驱动库
3. **长按检测**: 通过测量按键按下时长区分短按和长按
4. **状态机**: 通过布尔变量控制计时状态
