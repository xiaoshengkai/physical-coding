# 01 - 连接第一盏灯

## 实验目标

点亮树莓派上的第一盏 LED，学习最基本的 GPIO 输出控制。

## 硬件准备

- 树莓派 4B
- LED 1 颗（红色/绿色均可）
- 电阻 220Ω ~ 1kΩ（限流用）
- 母对母和公对公的跳线

## 连接图

```
[树莓派] 
 引脚11 (GPIO17)  ──── 红线 ──── 电阻①脚
                                      │
                                    [电阻]
                                      │
                        电阻②脚 ──── 黄线 ──── LED长脚 (+)
                                                       │
                                                      [LED]
                                                       │
                                    LED短脚 (-) ──── 黑线 ──── 引脚6 (GND)
```

### 引脚对照表

| GPIO 编号 | 物理引脚 | 功能 |
|-----------|----------|------|
| GPIO17    | Pin 11   | 输出（控制 LED）|
| GND       | Pin 6    | 地线 |

## 关键知识点

1. **GPIO 模式**: 树莓派使用 BCM 编号
2. **输出模式**: 设置引脚为输出模式
3. **电平控制**: HIGH = 3.3V, LOW = 0V
4. **电阻作用**: 限制电流，保护 LED

## 常见问题

**LED 不亮？**
- 检查电阻是否连接正确
- 检查 LED 正负极（长脚为正）
- 确认引脚连接牢固

**报错 "GPIO not available"？**
- 使用 sudo 权限运行
