# 02 - 呼吸灯

## 实验目标

使用 PWM（脉宽调制）实现 LED 亮度渐变，制作呼吸灯效果。

## 硬件准备

- 树莓派 4B
- LED 1 颗（红色/绿色均可）
- 电阻 220Ω ~ 1kΩ（限流用）
- 母对母和公对公的跳线

## 连接图

```
[树莓派] 
 引脚11 (GPIO17)  ──── 红线 ──── 电阻①脚
                                      │
                                    [电阻]
                                      │
                        电阻②脚 ──── 黄线 ──── LED长脚 (+)
                                                       │
                                                      [LED]
                                                       │
                                    LED短脚 (-) ──── 黑线 ──── 引脚6 (GND)
```

## 运行代码

```bash
python demos/02-led-breathing/led-breathing.py
```

## 关键知识点

1. **PWM**: 脉宽调制，通过快速开关实现模拟电压效果
2. **占空比**: LED 亮灭时间比例，控制亮度
3. **PWMLED vs LED**: PWMLED 支持模拟输出，普通 LED 只有高低电平

## 扩展思考

- 调整呼吸速度（修改 sleep 时间）
- 改变亮度范围（如 20%-80%）
